变量
biàn liàng
Variável Variable

Obs: também pode significar "inconstante/mutável" no sentido figurado.

我声明了一个变量来存储用户名。
Wǒ shēngmíngle yīgè biànliàng lái cúnchǔ yònghùmíng.
Declarei uma variável para armazenar o nome do usuário.
这个变量的值是空的。
Zhège biànliàng de zhí shì kōng de.
O valor dessa variável está nulo.

函数
hán shù
Função Function

Obs: termo formal; informalmente 功能 (gōngnéng) é usado para "funcionalidade".

这个函数返回用户的年龄。
Zhège hánshù fǎnhuí yònghù de niánlíng.
Essa função retorna a idade do usuário.
你调用了错误的函数。
Nǐ diàoyòngle cuòwù de hánshù.
Você chamou a função errada.

循环
xún huán
Loop Loop

Obs: também significa "ciclo" em contextos naturais e médicos.

这个循环会一直运行到条件为假。
Zhège xúnhuán huì yīzhí yùnxíng dào tiáojiàn wèi jiǎ.
Esse loop vai rodar até a condição ser falsa.
小心无限循环！
Xiǎoxīn wúxiàn xúnhuán!
Cuidado com loop infinito!

算法
suàn fǎ
Algoritmo Algorithm

Obs: usado tanto no contexto técnico quanto no cotidiano.

我们需要优化这个算法。
Wǒmen xūyào yōuhuà zhège suànfǎ.
Precisamos otimizar esse algoritmo.
这个算法的时间复杂度是 O(n)。
Zhège suànfǎ de shíjiān fùzádù shì O(n).
A complexidade de tempo desse algoritmo é O(n).
